# Catálogo de Filmes 🎬

Um aplicativo web moderno para busca e visualização de informações sobre filmes, utilizando a API do TMDB (The Movie Database).

![Catálogo de Filmes](https://i.imgur.com/placeholder.png)

## 🚀 Funcionalidades

- **Busca de filmes** - Encontre filmes por título
- **Visualização de detalhes** - Informações completas sobre cada filme
- **Modo escuro/claro** - Alterne entre temas para melhor conforto visual
- **Design responsivo** - Funciona em dispositivos móveis e desktop
- **Carregamento infinito** - Botão "Carregar mais" para ver mais filmes
- **Trailers** - Links diretos para trailers no YouTube
- **Informações detalhadas** - Elenco, diretor, sinopse e mais

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura da página
- **CSS3** - Estilização e animações
- **JavaScript** - Funcionalidades e interatividade
- **API TMDB** - Fonte de dados sobre filmes
- **Font Awesome** - Ícones
- **Google Fonts** - Tipografia

## 📋 Pré-requisitos

Para executar este projeto localmente, você precisa:

1. Um navegador web moderno
2. Uma chave de API do TMDB (gratuita)

## 🔧 Instalação

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/catalogo-de-filmes.git
```

2. Navegue até a pasta do projeto:

```bash
cd catalogo-de-filmes
```

3. Abra o arquivo `script.js` e substitua a chave da API:

```javascript
const API_KEY = "SUA_CHAVE_API_AQUI";
```

4. Abra o arquivo `index.html` no seu navegador.

## 🎯 Como usar

- **Buscar filmes**: Digite o título do filme na barra de pesquisa
- **Ver detalhes**: Clique em qualquer filme para ver informações detalhadas
- **Alternar tema**: Clique no ícone de sol/lua no canto superior direito
- **Ver mais filmes**: Clique em "Carregar Mais" no final da página

## 🌐 API TMDB

Este projeto utiliza a API do TMDB (The Movie Database). Para obter sua própria chave de API:

1. Crie uma conta em [themoviedb.org](https://www.themoviedb.org/signup)
2. Acesse suas configurações de API em [themoviedb.org/settings/api](https://www.themoviedb.org/settings/api)
3. Solicite uma chave de API para uso pessoal
