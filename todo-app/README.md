# To-Do List App

Um aplicativo de lista de tarefas moderno e responsivo, desenvolvido com HTML, CSS e JavaScript puro.

## 🚀 Funcionalidades

- Adicionar, editar e excluir tarefas
- Marcar tarefas como concluídas
- Filtrar tarefas (Todas, Pendentes, Concluídas)
- Limpar todas as tarefas
- Contador de tarefas pendentes
- Armazenamento local (localStorage)
- Modo escuro/claro
- Design responsivo
- Animações suaves

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e animações)
- JavaScript (ES6+)
- LocalStorage para persistência de dados
- Font Awesome para ícones

## 📱 Responsividade

O aplicativo é totalmente responsivo e funciona bem em dispositivos móveis, tablets e desktops.

## 🎨 Personalização

O aplicativo utiliza variáveis CSS para facilitar a personalização:

```css
:root {
  --primary-color: #4e54c8;
  --secondary-color: #8f94fb;
  /* outras variáveis... */
}
```

Você pode facilmente alterar as cores e outros aspectos visuais modificando estas variáveis.

## 📝 Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
3. Comece a adicionar suas tarefas!
